《人间修炼指南》

十号 2022-11-02 悟人生荒天帝

# 自序

作为一个风控从业者，我的世界观的基石之一是做每件事情都有代价；之二是总有外在风险需要防范；之三是我不犯错，最后就会赢。

三思而行嘛，这时候就要问下自己能真正的做到三思而行吗？不同的阅历所看待同一个问题的结果是不同的，重要的是珍惜当下，不负青春。

生活要小心翼翼，不要任性，抓住自己的核心利益，尽可能少犯错，然后做一些有利于自己的决策。

# 第一章 人间不需要值得，但生活需要

“不要丧失自己生活的主动权，不以外界的标准来衡量自己的价值。”

# 你不是废物，你只是还没有激活天赋

有一句话：人往高处走，水往低处流，这体现了事物发展的趋势。因为人是逐利的，所以都想去好的学校，而学校的运作也是一样的，它需要保持升学率就需要有优质的生源。这其实是双向奔赴。

好老师固然重要，但同样重要的是有天赋的学生，这其实也是很多顶级中学的核心运作思路，也就是所谓的掐尖，把全市乃至全省特牛的学生都收过来，然后打造品牌。

这些天赋就像智齿，终其一生都不一定会长出来，即时长出来了也有可能埋伏在牙龈里，不光看不见，派不上用场，还有可能带给你折磨。

形成了奖励闭环机制，从学习 -> 奖励反馈 -> 促进学习 -> 奖励反馈。侧面可以说明天赋是可以培养的，主要问题是应该怎么去培养天赋？话句话说，如何才能形成奖励闭环机制？

别人都一筹莫展的题目，他们随随便便就做出来了，这种成就感带来的是巨大的享受。反复获得超越别人的收货，反复强化这种认知，自然就会产生自信和兴趣，然后变的更强。

天赋没有出现在对的时间、对的地点、对的场合，那就等于零，甚至可能是负数。

所以当你很想要做某件事情的时候，就要去做。否则过了3天左右，也就没有激情想去做这件事了。如果去做了，有可能就会激活某一个天赋。如果不做，那么什么都不会被改变。

我们总是下意识地把天赋放一放、等一等，想在一切稳定下来以后再尝试自己的天赋。但那个时候我们可能会发现，曾经的天赋已经消磨殆尽。

其实天赋不会消融，它只是错过了哪个“适当的时间和适当的环境”。

无论这一次人生抓了副什么样的牌，都应该要将这副牌打好。珍惜当下，不抛弃，不放弃。不抛弃自己的人生，不放弃自己的选择。

人生的荒谬之处就在于，也许你是一株喜水植物的种子，上天却把你撒在了一片戈壁或沙漠里。

不确定的天赋，终将被确定的时间和地点，，固定成确定的人生。

但那个属于你的天赋，依然是你的保护伞、你的守护神、你信心的基石和命令的灯塔。

所以不要放弃，朋友。你不是废物，你只是还没有激活天赋。不要沮丧，你永远有你的世界宝藏。

# 年轻人到底要不要买房

因为第一套房是不具备金融投机属性的，你对于这套房子的需求是学区，是户口，是这个城市的归属感，是自己能够在这座城市落下脚、安下心，为了美好的未来去奋斗。

年轻人的第一套房，不是消费，而是资产固化，要搞清楚区别。

第一套房才可以无视风险买，因为你的核心诉求不依赖房屋涨价跌价，所以最大的风险于你而言可以被覆盖。

年轻人需要自由，需要无拘无束，没问题，但同样需要为未来的不确定性做一个风险兜底，第一套房，就是一个兜底。

必须综合考虑个体风险承受能力、资金成本、资金流控制、目标城市市场特色以及未来潜在政策风险来定制方案并不断调整策略。

面对第一套房都没有的年轻人，最需要的是先稳下来，再发展，再考虑之后的事情。

# 留学救不了内卷

所以，身为父母的职责是为了帮助孩子找到人生目标？还是那种自以为是的期盼强加给孩子呢？

喝鸡汤，打鸡血，从小卷到大，一个鸡娃闭环由此形成，焦虑感报表。导致的现象就是，孩子生理苦，家长心里苦。

在信息不发达的以前，出于慕强的心理，国内市场对国外学历充斥着盲目的迷信，同时由于难以分辨学校、专业的好坏，外国学历对国内大部分普通学历都有着强大的竞争优势。但现在，随着留学生的增多，留学生开始不再吃香，同时随着信息的发达，信息差开始被消除。

作为一个上司，我必然不可能会招一个比我还不能卷，同时还要一直教他做事的下属。我虽然不喜欢被人教做事，但我也不喜欢天天教人做事。

用留学来逃避内卷，说到底只是一种短暂的逃避，这种行为没有触碰到卷的本质。无法触碰本质，那必然无法带来根本性的改变。
卷的本质是什么？是资源有限，是分配机制有限，但每个人的欲望无限。
有欲望必然会增加对欲望的索取，而资源总量有限，无法满足每个人，因此恶性竞争就开始出现，内耗由此形成。

换一种说法，“卷”可能只是部分人无目的性的努力，这部分人不知道前方的目标是什么，只知道我需要比其他人更加努力。如果有清晰的人生目标，那么我为这个目标为之而奋斗怎么能称作“卷”呢？子非鱼，安知鱼之乐？

初见不知画中意，再见已是画中人。真正解决卷的问题，可能只能是放弃欲望了。

# 享受无聊，给生活留一些空白

有些内容只会让人短暂的快乐，有些内容却会让人持续的快乐。选择那些让你持续快乐的内容，正是人生厚度的持续积累。

有趣的事物和新闻围绕着我，我失去了无聊的机会，只能不停地不停地去摄入这些信息，就算吃饱了、吃撑了，也选不中页面上的那个“x”。

我不是玩APP，我是在被APP玩。

控制我的并不是外星人，而是我正在对生活失去掌控。

每个人都需要一段可以让自己真正静下来的时间，用于审视自己的内心或思考当下的这种生活的意义。

没有过多的娱乐元素，没有强烈的刺激，没有处心积虑地调用你的情绪的设计，一切都是生活的、自然的，也是无聊的。

真正的自由离我们太远了，毕竟普通人只有活着的权利而没有享受生活的权利。这时候我们应该要寻找一种平衡，一种在生活中苦衷作乐的平衡。

无聊的本质，就是你对自己时间和生活的掌控，而我们最缺的，恰恰是这种掌控。当你无法随时进入一种无聊的状态的时候，其实你的生活就是在走向失控。你曾经以为你是自由的，但是这个时候你会发现，其实你是不自由的。真正的自由是什么？是你可以随意无聊，也可以随意刺激，没有人逼你，除了你自己，没有人能掌控你的时间。自由是拒绝的权利。

如果大部分都有“无聊”这个需求，那么写一个极其无聊的游戏是否可以爆火呢？

无聊的东西，在这个信息过载的年代，其实是特别吸引人的。

就那样每天一点点地，为了一个目标，去忍受这种重复的、枯燥的学习过程，在无聊中去改变自己、提升自己，为了她的梦想去努力，其实就已经足够了。

我们并不需要去那么着急地摄入外界的信息，固然这些信息很有趣，但当它们开始让我们变得焦虑，变得神经衰弱的时候，它们就不再有意义。

我们需要的，其实是随时能看到另一个世界的选择权。那时你才能意识到，你真正追寻的不是有趣或者无聊，而是在这个变化的世界中，保持本心的自己。钱塘江上潮信来，今日方知我是我。如是而已。

# 用利益思维选择学校和专业

大学和专业选择的核心思路只有一个，那就是围绕就业和赚钱，一定要面向这两个目的来做，其他东西都是虚的，你得想清楚你的生活。

很多年轻人会觉得靠家里的优势达成目的是一件非常丢人的事情，这个想法是幼稚的，小朋友才讲究单打独斗，社会人都是结果导向，能用优势兵力解决绝不单挑。

初入职场，缺的是引路人

如果你家里人只是普通地深耕某个行业，一家人都是做铁路的，都是做电力的，都是做教室 的，都是做律师的，都是做金融的。那你选择类似的专业可以得到非常专业的指点，而且是不带有利益算计的毫无保留的指点，这对于一个初入职场的年轻人而言，就是开挂。

城市和学校决定了我们将来毕业时的关系圈和人脉圈，这是一个大学生刚毕业时的起步资源，一旦换城市就业，会直接浪费掉大学时候积累的资源，非常不划算。

工作是为了赚钱，赚钱是为了快乐，如果直接失去了快乐，那还弄个啥。

不是学了金融才有钱，而是有钱人喜欢让自己孩子学金融，金融可以合法把家里的优势资源变现。

互联网时代了，信息根本不是不够，而是看大家肯不肯动脑子。以及愿不愿意执行。

大多数人只是喜欢玩这些东西，本质上让你沉迷的是玩乐享受，以及玩乐享受带来的某种优越感。

真让你踏踏实实从专业角度来执行这件事，你可能会痛不欲生。

对所有行业的浪漫幻想本质源于对行业的无知。

谋生的职业，需要的是专业精神，并且需要功利地看待，而热爱是没有功利性的，这是不可避免的矛盾，无法解决。

一个坐办公室的白领，随时可以选择去做一个卖煎饼果子的，他有选择权。

但是一个初中毕业卖煎饼果子的人，可能永远也没发选择做一个拥有五险一金天天吹空调工作体面不用担心城管的白领，他们可能只能选择去搬砖、去送外卖、当保安、上流水线，他们没有回到办公室的选择权了。

在选专业的时候，一定要考虑到向下兼容性，尽量选择有门槛的专业。一是有就业优势，而是给自己留了转 其他行业的后路。

实习经历大于专业比赛大于学习成绩。

工作的目的就是赚钱，赚钱的目的是让自己有选择权。

# 第二章 不要温和地走进那个职业

# 职场薪资倒挂的背后，是利益最大化

踏入社会的第一步，就请先把学校里叫你的规矩丢掉，因为你没有后盾了，社会上比拼的不是这些。

如果一个东西听起来违背常理或者直觉，但又非常流行，那么只有一个答案，那就是这东西确实有用，而且好用。

在现实职场中，很多管理者权利扩大，是靠自己管理人数的增加来实现的。

如何才能快速撬动权力膨胀？

最简单的方法是提需求招人，而且是快速招人。很多哦空降管理者最爱干的事情就是招人，不招人他们都不知道干什么了。

你以为管理者不懂，实际上他们比谁都懂，只不过大家的利益不一致而已，人都是趋利避害的。

管理者最怕的不是错，而是弱。

所以杀鸡儆猴也是一种策略，要树立自己宁可去外面花贵的钱也不愿意屈服的形象，因为这次退一步，下次就要退更多步，最终导致人心不平衡，团队失控。

很多老人真的没有新人优秀，他们最大的优势不是能力，而是比较融洽的同事人脉，以及对于公司内部流程的熟悉程度。

最起码每年都要去同业那里面试一下，跳不跳不重要，起码你要知道市场价，也要知道自己的价值。说不定你拿了一圈offer发现价格还不如自家，或者拿不到满意的offer，那说明你现在的公司没有亏待你，你就只值这个价，摆正心态，好好上班，现在是市场经济了。

记住，即使被别的公司拒绝了，也尽量向面试官或者HR诚恳请教自己被拒绝的原因，多数人在没有利害关系的前提下是愿意和你分享一些看法的，这些看法都有利于你完善自我。

我们出来工作就是要赚钱。所谓的职业规划，说白了就是看赚短期的钱还是赚长期的钱，归根结底重点还是在赚钱。

没有财富自由，就没有思想自由，所以我认为各位追求钱是非常正当的事情，我自己也是这样的。

# 搞钱没问题，但只盯着钱就比较蠢

现实中大部分人其实是没有资格选择的，当一个年轻人刚刚毕业，拿着不多的钱到一个新城市，他需要做的只能是快点找一份工作，来赚出下个月的房租。

他们不是不能吃苦，更不是不肯吃苦，而是他们一直在拼命努力，最后却因为信息差被收割。

很多人习惯在一条固定的赛道上拼命加速，结果到了十字路口就蒙。

在这个“做题家”世界里，人人都知道要考985、211，但是并不是人人都记得考完试还要选专业。

钱的总数是单价X时间，不能光看单价，也要看时间。

要想长期持续的搞钱，就需要有长期的规划，工作是有性价比这个概念存在的。

我学历不行，我得给自己加标签，这种能够咸鱼翻身的机会，倒贴钱也要去，我和家里也这么说的，家里愿意支持。

当你选择一个工作的时候，你先问问自己：三年后这份工作能带给你什么？你的三年工作经验能在市场上换多少钱？

只要有的选，就一定要仔细调查自己备选公司的状况，钱很重要很重要很重要，但钱绝对不是唯一标准。

你干的职位，是抓个人就能做的消耗岗位，还是有哦升职空间，能够带给你更多技能成长的核心岗位？如果干的不顺心要跳槽，这家公司能给你带来什么样的加成？这些信息都会成为择业时的关键要素。

在和资本搏斗的过程中，“韭菜”最大的问题并不是资产问题，而是信息问题。

先锁定自己想要前进的方向，把这个方向上的信息都搞透，在局部地区获得信息优势，让自己能反过来占资本便宜。

资本要你的劳动力，你就去蹭资本的附加值，从资本身上捞到自己想要的标签，然后哦卖掉这些蹭来的附加值，这才是搞钱的正道。

# 为何有的职业后期不给力

特别留情的企业，要么本身就是垄断，要么不需要赚钱，要么已经挂了。

及早放弃幻想，不要相信公司是家，给自己准备第二条道路才是王道。

很多互联网小公司其实就指着几个项目吃饭，一个新项目完了以后未必还能有开发下一个的机会，不节约成本说不定明天就倒闭了。

这个世界上只有你自己才能为自己考虑。

低门槛不代表低上限，往往越是低门槛和速成的行业，能脱颖而出的越是大牛，比如销售，看似谁都能干，但能干好的绝对不是一般人。

个人水平的差距在被黑箱无限拉平，体力反而变成了主要的竞争力。

技术工种到最后拼体力，只能说明技术本身没有那么高的门槛。

并不是说传统行业都是大后期，老师和医生这样的职业之所以能够后期发力，是因为这样的职业吃的资源多，越到后期替代成本越高。

这个道理其实适用于任何一个职业，甚至任何一个个体——你吃下的资源越多，要替代你的成本越大，你就越有价值。

很多互联网从业者并没有清晰地意识到自己是占了职业的便宜，还以为这辈子都能拥有远高于其他行业的收入水平，于是养成了把后半辈子的钱也一起花掉的消费习惯。

他以为自己的人生是一柱擎天，但其实只是抛物线。本质上，这才是很多人陷入中年危机的根源。

无论从事什么行业，都要多吃资源，增加个人价值，提高企业的替代成本。

虽然是零件，但中枢轴承的待遇和外围齿轮的待遇终究是不一样的。

有时候社会竞争的本质逻辑，就是比拼谁能在短时间吃下更多的资源，然后快速消化，再去抢更多的资源。这是一个吃得快跑得快的游戏。

要知道自己的职业是前期职业，所以要趁年轻的时候多存钱，降低物欲，增加抗风险能力。

时刻保持警惕，时刻给自己留一条后路，不要把鸡蛋装在一个篮子里。

# 为什么现在的年轻人不愿意讨好领导

这件事情做了是不是有好处？

这件事情不做是不是有坏处？

如果没有前两者，那我喜不喜欢做这件事情？

遇事多在自己身上找原因，是一个非常有效的看问题的方法。

大家讨好的是你给的好处，而不是你，不要自我感觉良好。

一切潜规则的前提都是好处，这个好处不一定是特别大，但一定是双方都认可的价值。

管理者的核心诉求是带领团队尽可能高效低成本地达成目标。

# 录音笔是职场的最后尊严

我们体面人，都是用录音笔的。

在离职这件事情上，要学会做每件事都要假设最坏的结果，绝不相信任何人的任何一句话，这就是录音笔精神。

# 第三章 只要不犯错躺着都能赢

面对一切超出正常生理需求和社交需求的消费，统统拒绝，反复默念“我很穷，我没有钱，只要我足够穷，智商税就追不上我”。

# 健身房为何频频跑路

因为人性就是贪嗔痴懒馋。惰性是刻在人基因里的东西，葛优躺才是人类最通用的广播体操。任何违背让人贪嗔痴懒馋的东西，都注定是难以维持的。

间接性打鸡血，持续性混吃等死才是人类的最爱。

足够有钱，足够有闲，足够大的包容性，这些是健身房存在的土壤。目前这片土壤，是非常贫瘠的。

一家公司没有利润不一定死，但是没有现金流则一定会死，利润只是公司的脂肪，现金流才是公司的血液。

健身房行业有个比较有趣的说法，开业半年以内的健身房要小心，开业五年以上的健身房要小心，开始卖终身卡的也要小心。
