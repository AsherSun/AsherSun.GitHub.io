    • 金钱的重要性，一直都是我们赋予它的。
      • 如果你有财务问题，金钱就会变得尤为重要。
    • 希望是一种智能镇定剂，一种绝妙的自我欺骗。
    • 我们迎接障碍，因为我们不认为生活会优待我们，不认为自己已经强大到能够实现目标。
    • 乐观主义让你看到事物的积极面，而自信给你一种战胜事物黑暗面的信心。
    • 自信指的是，一个人在自身经历的基础之上，知道他遇事可以依靠自己
    • 你的财物状况应该成为一个你不惧怕任何阻碍的证明。
    • 一个没有自信的人，只是一个一事无成、一无所有的无名小卒。
    • 你的思维方式铸就了你今天的样子。这样的思维方式却并不能让你变成你想成为的样子。
    • 财务问题总是会对其他几个领域造成消极影响。
    • 永远不要为了金钱去做一些无趣的事情。
    • 做个选择：你是想拥有一台赚钱机器，还是穷其一生当一台赚钱机器。
    • 将你的事业建立在你最大的爱好之上。
    • 整天工作的人是没有时间来赚钱的。
    • 每天问自己的问题
      • 认识你自己，弄清你真正感兴趣的是什么？
      • 想一想你应该如何用你的爱好来赚钱？
    • 不是你自己规划你的生活，就是别人来规划你的生活。
    • 我们的价值观和目标必须协调一致，否则我们便会处于矛盾之中。
    • 梦想：
      • 如果你拥有无限的时间和金钱，你会做什么？
    • 价值观
      • 对你来说真正重要的东西是什么？
      • 为什么这些东西很重要？
    • 目标
      • 你想要成为什么样的人？
      • 你想要做什么？
      • 你想要拥有什么？
    • 策略
      • 你具备能协助你获得自己想要的东西的知识、能力和计划吗？
    • 只有当你清晰地确定知道自己的价值观之后，你才能掌控自己的生活。
    • 成功意味着得到你想要的东西，而幸福意味着热爱你所拥有的一些。
    • 经常思考这些问题：
      • 这应该如何为我所用？
      • 我如何快速采取行动？
    • 知识运用于实践，才能获得真正的力量。
    • 你一生中能否有所作为，并不直接取决于你铁一般的行动准则，而是与你的梦想、目标、价值观和策略直接相关。
  • [x] 5月8日看完第二章《责任意味着什么》
    • 给你责任的人同时也给你了权力 ——韦恩·戴尔博士《如何成为跨越极限之人》
    • 人们并非要对所有的事件都要负责，但你总是要对自己对事件的判读和反应负责。
    • 对于自己的遭遇，我们能够自己决定应该做出怎样的应对。
    • 人们需要对自己的内心反应负责——对我们的态度和我们表现负责。
    • 以争吵应对争执，以冲突应对挑衅，绝非最佳的问题解决方式。
    • 你把责任推给了谁，也就给了他权利。
    • 如果你喜欢掌控自己的生活，那么就将责任揽到自己身上。
    • 在我们做出行为决定之后，其所造成的后果也随之而来。拾起木棍的一端，他的另一端也随之起来
    • 我们对事件后果的判断和反应，便是我们的责任所在。我们想要自己掌控将来事态的发展，还是让我们的错误及其后果掌控将来的事态？
    • 我们通常都是一点一点地发生改变，这些变化难以察觉。
    • 人们可以通过扩展可控领域来积累你的财富。
    • 如果总是谴责客观因素和他人，我们就相当于自动放弃了拥有控制权的机会。
    • 如果我们习惯于在每一次战胜挑战之后都马上迎接新的更大的挑战，那我们的可控领域便会快速地得以扩展。
    • 我们虽然需要安全感，但同样也需要冒险和改变。
    • 每一个问题也都为我们带来成长的机会。
    • 逃避问题不会使人生活富裕。
    • 提问质量决定我们的生活质量。
    • 我们首先要问的不是“为什么”，而是“怎么做”。“怎么做”寻找的是解决方案，“为什么”寻找的是托词。
    • 只关注我无法做到的东西并不会给我带来金钱和好处。相反，我必须更多地关注我会的东西。
    • 正确的提问方式应该是：“我会做什么？我知道什么？我有哪些机会？”。
    • 当你做决定时，你应该问：“是否”。
    • 正确的决策顺序应该是：
      • 我为什么要这样做？决定“是否”要做。
      • 我应该“怎么做”？出现下一个困难的话，你如何解决？一直牢记不放弃的理由。
    • 很多时候，不承担责任显得方便又舒适。我们之后需要付出的代价是极高的。因为我们会被别人玩弄在手掌之中，只能按照别人为我们谱写的剧本来生活。
    • 我所理解的成功就是成为最好的自己。我全力以赴时，会感受到前所未有的活力。
    • 你必须避免金钱成为你生活中价值很高的东西。因为，你必须承担责任。
    • 明智的行为会伴随我们一生，使我们感到快乐，救我们于危难；同样地，愚蠢的行为也会伴随我们，是我们备受折磨，烦恼不堪。
    • 责任意味着：没有任何东西能改变你的态度和品格，因为你是按照自己的本性对事情做出反应。你可以决定自己以哪种方式去生活。这种态度使你能够生活幸福，并成为最好的自己。如果你勇于承担责任，那么你就能在未来变得更好。
  • [ ] 5月8日看完第三章《100万欧元是难以企及的高度吗》
    • 大多数人都高估了自己1年内能做到的事情，也低估了自己10年内能做到的事情。——吉姆·罗恩《雄心的力量》
    • 发生变化的5个层次
      • 第一层次：你意识到自己对现状感到不满意。为了改变现状，你开始采取行动。
      • 第二层次：期待的结果没有出现，你意识到，仅仅只是采取行动还远远不够。你采取的行动还必须以解决困难作为目标导向。
      • 第三层次：学到的技巧对你起到了一定的帮助。
        • 想要成为你想成为的人，那就需要投入时间了。
      • 第四层次：在你与自身和他人打交道是，仔细思考一下你的世界观。
        • 同黑白眼镜相比，一定也有一些有色眼镜，使我们自己和他人更加幸福。也有一些眼镜，使我们看到的是机会，而非错误和陷阱。
        • 一念天堂，一念地狱。
      • 第五层次：通过改变对自身的认知，我们可以带来巨大的变化。
        • 将自己看做销售人员还是专家，做出不同的决定，所产生的效果是不同的。我们的自我认识，最终会成为自我实现的预言。
        • 我要在每一次挫折中看到积极的一面。我确信，一切事物都有其积极的一面。
    • 奇迹发生在5个层次之上
      • 有些人为了解决问题，会主动去做所有必要的事情。
        • 在这样的人身上，“困难”这个词并不会引发出绝望的情绪。他们也不会悲叹：“不，为什么这件事偏偏要发生在我身上。”
      • 原理很简单：
        • 第一层次：奇迹之所以发生，是因为我们为此付出了努力。
        • 第二层次：迎接困难并去解决问题有助于技能的掌握。
        • 第三层次：你应该成为一个人物，一个因为自身能力强大，所以别人主动来接近你的人物。
        • 第四层次：困难不是问题。人生的意义不是为了在困难面前躺平而存在的。
        • 第五层次：身份，一个“没有问题的人”。
      • 干坐着等待奇迹的发生，是一种愚蠢的行为，就好像一个运动员坐在家里的电视机面前观看奥林匹克运动会，希望自己能获得一枚金牌。
    • 书籍
      • 引发奇迹的4个法则统称为：持续不断的学习和成长。
        • 持续不断地学习和成长意味着，感受到自身源源不断的活力，也意味着你能成为最好的自己。
      • 一个人是他读过的所有书的总和。 ——一位智者。
      • 在当今社会中，我们只需要花几个小时，就能读到前人花多年时间总结的经验和研究之精髓。我们不必再去犯前人犯过的错误。
    • 你的个人成功日记
      • 将自己看作一个足够重要的人物，开始记录关于自己的日记。
      • 信心不足的人为了保护自己不敢承担风险。然而，不敢承担风险的人，将永远无所作为，只会一事无成、一无所有。
      • 一本成功日记，比起其他任何东西，能更系统更有效地帮你建立起自信。
      • 不方便或是没有兴趣，常常让我们不愿采取下一个决定性的步骤。然而，不方便只是一种托词。实际上，是我们不相信自己能取得成功。
      • 通过每天写成功日记建立自己的信心：
        • 我们的自信心决定了我们是否敢于冒险。
        • 不去冒险便不可能获得成长。
        • 我们的思想意识、我们受到的教育、我们周围的环境，总体来看不利于我们自信心的培育。健全的自我评估意味着自信。成功日记会使你意识到自己有多优势。
        • 写日记时，你学会了去注意自己的长处。
        • 在每一段时间之后，你就会开始发现自己的长处并想：这件事我可以把它写进我的成功日记里。
        • 我们的期望决定我们会获得什么。我们的自信又决定我们的期望。
    • 讲座
      • 我们能在同一时间听到、看到、感受到并且经历许多东西。我们的感官接受的刺激越多，获得学习效果就越好。
      • 我们为教育所付出的代价，与我们为无知愚昧所付出的代价是无法相比的。
    • 榜样
      • 从出生那一刻起，我们就通过模仿来进行学习。
      • 近朱者赤，近墨者黑。
    • 要创造奇迹，你需要勇气
      • 以为目光远大的思想家也需要这样的信念。这其中的艺术便是，不要让自己丧失勇气。
    • 没有哪个奇迹的发生不需要冒险
      • 风险是什么？
        • “当你不害怕的时候，斗牛对你来说算不上什么，当你感到害怕，而不去斗牛那也没有任何意义。但当你害怕斗牛，而仍然去斗牛，你就干了一件有意义的事。”
        • 所有有价值的东西，在最开始都可以把我吓死。
      • 我们不应该以他人来衡量自己，而应该与自己做纵向对比。
      • 读一读下面的句子，仔细考虑自己是否乐意承担风险。
        • 我们常常忘记，顺着我们目前为止走过的道路继续前行，同样有风险。这条路给我们一种熟悉感，但也不会更安全。
        • 生活就是一场比赛。不去冒险的人不可能获得成功。
        • 在这个星球上，没有“必胜”，只有“机会”。
        • 立即采取行动！因为你永远不能为成功做好完美的准备。
        • 世上唯一从来不犯错的人，是从来不做任何事情的人。
        • 不确定性带来的痛苦，比确定的痛苦要强烈得多。
        • 每一次行动计划都有风险，都需要付出代价。但比起无所事事的舒适，它们的长期风险和付出的代价要小得多。
        • 如果你害怕失去，你永远都不会获得。
        • 去冒险吧，你不可能从地面摔下去的。
      • 想让某些事物为了我们而改变，我们自己就必须先做出改变。
      • 每一次改变对我们来说都是一次冒险，因为我们需要以某种形式离开自己熟知的环境。
      • 奇迹总会发生在有变化的地方。但你必须准备好，离开你的舒适区域，去冒险。同时你也需要好运。
    • 你真的需要好运吗？
      • 什么不是好运？好运不是一种不通过努力，光靠期待就可以获得的东西。
      • 根据研究结果，惊人的好运通常只是多年准备的结果。
    • 好运是如何产生的
      • 好运都是以机会的形式敲击人们的房门。
      • 好运不会等待慢吞吞的人，我们必须马上采取行动。
      • 好运组成的部分：
        • 储存资本；
        • 识别机会；
        • 果断地作出决定并采取行动。
      • 一个拥有极大好运的人，通常是自己做好准备并且主动冒险的人。
      • 好运也只是多年准备的结果。
      • 如果你主动承担责任，那么你的好运就是可以期待的。每个人都会得到属于自己好运的机会。
    • 创造奇迹容易吗？
      • 很简单的事情，如果需要坚持不断地做下去，也不容易。
      • 机会不会将时间浪费在没有准备的人身上。
      • 每一次准备都开始于人们内心态度的转变。
      • 愿意承担责所有的责任，对奇迹的发生是至关重要的。

    • 比赛是消极防御还是天然进攻，两者之间有着天壤之别——丹尼尔·S.佩纳《论融资》
    • 不同的选择差异，是我们的信念造成的。
    • 穷人对财富没有明确定义
      • 生活就像一家邮购商店。寄来我们订购的东西。
      • 定义财富的三个条件：
        • 必须知道明确的数字
        • 写下这个数字
        • 将它转换成画面
      • 想要真正的积累财富，就应该潜意识采取同步行动。
      • 如果不能让自己置身于这些过度刺激之中，那我们也不必惊讶于自己一事无成。
      • 只要我将精力集中于这个目标，恐惧和怀疑就无法控制我。
    • 穷人的财富目标摇摆不定
      • 目标越远大，你需要做的改动就越少。
      • 一旦我们无法将目光集中与目标，怀疑和恐惧就产生了。
      • 在设立一个大目标的情况下，问题的出现就无法完全阻挡你看见目标的视线。你清楚自己为了什么而努力。
      • 目标越大，问题相对来说就较小，即使它们可能看起来无法克服。
      • 为自己设定自己有生之年无法实现的目标。
    • 穷人从不将财富看成绝对必需品
      • 我要登上这座山，你只会看到我在山峰上招手，或是看到我的尸体躺在谷底。
      • 你有能力使你的目标成为绝对必需品。
      • 不实现这些目标你将无法幸福。一成不变的生活对你来说是巨大的痛苦。你一定要实现自己的目标。
    • 穷人做不到坚持到底
      • 如果我们为自己设立极限，那么我们就会去达到这个极限。
      • 下定决心之前，你应该首先弄清楚，你是否真正的想要实现这个目标。
    • 穷人不去承担责任
      • 如果将精力用来提问“为什么”，那么我们就会找借口。而提问“怎么做”的人，则将精力放在寻找解决方法上。
      • 永远不要让别人替你承担责任的情况出现。
      • 你的成功仅仅取决于你个人，你对此负有责任。
      • 你如果认为成功是不可重复的，就是去了提升自我价值的机会。
    • 穷人不准备付出100%的努力
      • 隐藏着自我欺骗、恐惧和缺乏自我价值感的两句借口：
        • 将来我也会为现在拥有的一切感到满足
        • 假如我能全力以赴，我就会成为最优秀的。
      • 我们对于当前的成就存在感激之情，这是美德。但对于当前拥有的东西，将来也感到满足，这就违背了人类不断成长的需求。
      • 我们全力以赴时，能够获得最大的满足感。
      • 无论是谁，将最后一个可能的借口都摒弃掉，必将能取得成功。
    • 穷人缺少一个好的教练
      • 只有教练才可能根据自身经验，在尽可能短的时间内将运动员的才能激发出来。
      • 一位好的教练也能长期对你施加压力，促使你付出110%的努力。
    • 穷人只关注自己的劣势
      • 最先关注自己劣势的人是不会得到财富的。
      • 同自身劣势相抗争的人会花费大量的精力，这样一来，他最终也就是一个平庸之人。
    • 你的优势使你富有
      • 你不能忽略你的劣势，但你也不应该同劣势相抗争，因为这并不能使你变得富有。因此，你应该为你的劣势找到一个正确的解决方案。
      • 去为你的劣势找到解决方案，去为你的优势找一个教练吧。
    • 变得富有的代价有多大
      • 与生活富裕相比，长期的财务危机更可能给家庭生活蒙上一层阴影。
      • 与坐拥金山相比，家徒四壁更可能使人去犯罪。
      • 代价是：花费时间去做你想做的事情，去精通你的技能树。
