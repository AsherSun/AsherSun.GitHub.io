# IT从业者聊投资

# 书籍、论坛

  • 简七理财 ？是一个论坛吗？
  • 雪球财经
  • 巴菲特 《致股东的信》
  • 芒格 《穷查理宝典》
  • 《穷爸爸富爸爸》2月份读完

了解 上市公司财报、商业模式

  • 为什么要去了解上市公司财报、商业模式？
  • 首先要对投资比较感兴趣

# 基金选择需要注意的点

  • 选择自己比较懂、比较熟悉的行业

    指数基金

  • 关注基金经理一直以来的整体水平

    从业经历、时间、投资了哪些行业以及公司？

    收益率、平均延放

  • 关注基金规模的级别

    新发基金少碰

    新的基金经理发放少碰、观望

    十亿级别、百亿级别的规模可以规避一些雷

  • 尽量规避新发行的基金

  • 关注基金背后公司及其持有比例

  • 基金的分红情况

    继续定投

    分红提现

  • 关注费用：申购费、赎回费、管理费、托管费

    短期购买，赎回费会比较贵

    持有时间越长，管理费可能越贵

    持有时间越长，赎回费会越低

# 投资策略和原则

  • 指数基金采取定投的策略，跌的很凶的适合，就手动买入

    设置定投规则，比如：每天定投多少、每周定投几次？

    如果跌的很凶，可以手动买入

    如果涨的起来，可以停止定投

  • 闲钱投资

    一定要闲钱投资

    要准备好3 ~ 5年的生活费、意外情况的资金储备、买入一些保险

  • 设置期望收益，道理自己的期望值，需要卖出，然后继续慢慢的定投

    到了期望收益值，就卖出

    如果连续涨的很凶，已经到了收益点，但是定投金额少。可以考虑持有，如果定投数额比较大，比如一万、两万。可以选择卖出。继续慢慢定投

    中间基金经理变化，需要关注。去了解新的基金经理的相关背景和资料，再结合情况选择是否继续持有

# 影响投资的因素

  • 心态要平稳，可以看看投资心理学

    追涨杀跌，从投资角度来看是不合理的。应该要追跌杀涨

    心态要平稳

  • 不要过度频繁交易

  • 需要独立思考和分析上市公司

    不要听一些消息

    要独立分析财务报表、业务，是否有新的业务？投入的研发？

    是否有投入收益较大的业务研发

    公司的懂事、领导人是否可以让你放心？

# 程序员在投资领域的优势和劣势

  • 每天对着电脑，能及时获取投资咨询
  • 收入总体尚可，能为投资提供较好的资金流
  • 投资者也需要持续学习，有共同点
  • 思维不受传统金融规则束缚，比较容易掌握线上交易的APP
  • 对数学和计算公式敏感
  • 牛逼的人可以去做量化交易
  • 程序员工作较忙，容易错失短线交易买入卖出的时机

# 个人生活资金控制

  • 早餐10元
  • 晚餐20元
  • 餐食费：900
  • 其他花费：124
  • 房租、水电费: 2000
  • 还款：4600

基本花销：4600 + 3024 = 7624

每周定存75元，每月定存：5 * 75 = 375

如果个人工资是12000(到手工资，不包含5险一金)，那么减去基本花销：12000 - 7624 = 4376

那么我每月可动的理财金额最大限度占收入的百分比是：4376 / 12000 = 36.46666*%

每月存钱最低：2376，那么每月可动理财金额是：4376 - 2376 = 2000

每月存钱 - 每月定存 = 2001元。每月最少存钱：2001元

那么每月占工资收入的百分比是：16.66666*%
